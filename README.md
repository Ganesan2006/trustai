# TrustAI – Document Q&A with Smart Citations

TrustAI is a RAG (Retrieval-Augmented Generation) system that allows you to upload documents and ask questions with inline citations.

## Features

- 📄 Upload PDF, DOCX, TXT, MD files
- 🔍 Semantic search with Qdrant
- 📚 Inline citations with source cards
- 👍 Like/Dislike feedback
- 👥 Role-based access (Company, Department, Team, Personal)
- 🌓 Dark/Light theme
- 📱 Mobile responsive

## Tech Stack

- FastAPI (Backend)
- PostgreSQL (Neon)
- Qdrant (Vector DB)
- Supabase Storage (File storage)
- HTML/CSS/JS (Frontend)
